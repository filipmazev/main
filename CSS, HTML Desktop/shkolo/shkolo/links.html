CSS | ДЕМОНСТРАЦИЈА

Пред CSS, речиси сите презентациски атрибути на HTML-документите беа

содржани во HTML-ознаката. Сите бои на фонт, стилови на позадина,

порамнувања на елементите, границите и големини мораа деталлно да се

опишат, честопати постојано, се во рамките на HTML. CSS им овозможува на

веб-дизајнерите да пренесат поголем дел од тие информации во друга датотека,

стилскиот лист на CSS, што резултира во значително поедноставен HTML.

На пример, заглавја (h1 елементи), под-наслови (h2), под-под-наслови (h3) итн.,

се дефинираат структурно со користење на HTML. Изборот на фонт,

големина, боја и акцент на овие елементи е презентациски дел, односно

CSS делот. Пред CSS, авторите на документите кои сакаа да доделат такви

типографски карактеристики, да речеме, за сите наслови на h2 мораа да повторат

HTML презентациска марка за секое појавување на тој наслов ( секое h2 ).

Ова ги направи документите покомплексни, поголеми и посклони на

грешки, правејки ги тешки за одржување. CSS овозможува раздвојување на

презентацијата од структурата. CSS може да дефинира боја, фонт, порамнување

текст, големина, граници, растојание, распоред и многу други

типографски карактеристики.
