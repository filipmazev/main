CSS

Секој css документ завршува со '.css', името на документот е она кое го користиме за да го поврземе со html документот.

За да го поврземе HTML документот со CSS документот, веднаш по 'head' делот од HTML ја пишуваме командата :

< link rel="stylesheet" type="text/css" href="(име на документот).css" >

.

Бојата се менува со командатa color: (боја | пр: white или пак #f2f2f2 во hexa-decimal ).

HTML

Секoj html документ завршува со '.html' по името. HTML е делот во кој го пишуваме текстот и генерално

целиот материјал на веб-страната. Додека пак CSS го користиме за креирање на стилот, односно изгледот на страната.

Во HTML имаме неколку вида на текст, пример 'h' (header) типот, кој оди од најголемо h1, па се до h6, најмало.

Текстот до сега беше во 'p' ( < p > ) типот, односно paragraph или параграф, тука се пишува повеќето од текстот.

.
