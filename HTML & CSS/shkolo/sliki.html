CSS

Со помош на CSS можеме да ја разубавиме нашата веб-страна преку сликите кои ги имаме внесено ( поврзано ) во HTML делот.

До сега на секоја од страните позадината беше .gif file поврзан во CSS со следната команда

< background-image: url('име на слика.формат');

Кога внесуваме слика мораме да обрнеме внимание на следното:

1. Форматот на сликата ( jpg, png, gif... итн. ) | 2. Локацијата на сликатта ( мора да е во истиот фолдер каде што се и css и html документите)

3. Доколку сликата не е симната локално и е поврзана преку линк мораме да запазиме на тоа дали линкот е точен и дека е од самата слика,

овој линк се добива преку следните чекори: десен клик на сликата која сакаме да ја искористиме по што кликнуваме на 'copy image address'.

Слика од линк се поврзува со командата: < img src="линк" > во HTML или со url(линк) во CSS | исто принцип важи и за локално симнатите слики,

но на местото на линкот би стоело името на сликата ( пр: imageExample.jpg или пак image13.gif )

.

сликата е зачувана локално под името 'slika6.gif' и е поврзана со командата: < img class = "image1" src="slika6.gif" align="middle" > | CSS дел :

width: 40%; ( големина на слика ) | background-color: white; ( боја на рамка ) | border: 1px solid #ddd; ( боја и големина на надворешен дел од рамка ) | border-radius: 4px; ( радиус на рамка, во пиксели )

padding: 10px; ( дебелина на рамка ( односно оддалеченост од сликата ) ) | box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2), 0 6px 20px 0 rgba(0, 0, 0, 0.19); ( сенка )
